ls ->Lista o que há dentro do diretório
cd -> Muda de diretório
pwd -> Mosta o caminho absoluto de onde estou
ls -a -> Mosta as pastas ocultas
cd .. -> Retorna uma pasta a cima da que se encontra
mkdir -> Criar nova pasta
cat -> mostra o que tem dentro do arquivo consultado
rm -> remove arquivo ou diretório
rm -rf .git/ -> remove de forma recursiva e forçada todos (tomdiretóriose muito cuidado com este comando, se o fizar em pasta errada, terá problemas)
